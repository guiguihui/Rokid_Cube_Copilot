# 测试图片（绕过摄像头）

把 6 张「单面」照片放到这个目录，命名如下（每张拍一面，色块尽量填满画面中央）：

- `U.jpg` 顶面（白中心）
- `R.jpg` 右面（红中心）
- `F.jpg` 前面（绿中心）
- `D.jpg` 底面（黄中心）
- `L.jpg` 左面（橙中心）
- `B.jpg` 后面（蓝中心）

然后在扫描页对每一面选「测试图片」即可走和拍照一样的识别管线（颜色来自你的照片，更准）。

也支持 `.png`/`.webp`；若用其他扩展名，改 `lib/testImages.js` 的 `TEST_PATHS`。
若 Craft 里 fetch 读不到本目录，告诉我，我把图片烘焙成 base64 填进 `TEST_DATA`。
